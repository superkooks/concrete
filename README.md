# Concrete Alarm System
A complete alarm solution for the home automation era.
Currently under development.

## Features
- Tamper detection using double end-of-line resistance monitoring
- RS485-based protocol for keypads and expansion boards
- Support, including charging, for 12V lead-acid backup battery
- Ethernet connectivity for talking to Home Assistant
